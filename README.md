# Product Database
**Product Database** - представляет собой базу данных, в которой хранятся данные о продуктах и категориях. В базе данных реализован SQL-запрос, который возвращает все пары "Имя продукта – Имя категории", даже те продукты, у которых нет категорий.

### Инструкция по запуску
1. Постройте проект, выбрав опцию **Build**.
2. Опубликуйте базу данных используя опцию **Publish**.
3. Запустите запрос находящийся в файле **`SelectProductCategories.sql`**.
